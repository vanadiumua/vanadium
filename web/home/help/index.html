    Help

    Welcome to Vanadium, a utility records access system. Vanadium is a web-application designed to allow you to access older utility records that would otherwise be lost or inaccessible due to legacy, deprecated software packages.

    Vanadium stores all of your data in a secured relational database. If you need to carry out bulk operations, ask your system administrator about how to access the raw data through phpMyAdmin, which will allow you to download the tables in their normalized form.

    Printing

    If you are only looking up individual queries, you can use the pretty interface. You can also print from any page, using your web browser's built-in print functionality. However, we suggest using the print button if you are generating a more detailed report. This will format the webpage in a print-friendly manner.

    Search Queries

    You can search in one of three ways: by searching all searchable fields, by searching specifically by account number or address, or by displaying all of the entries in the database.

    When you search all searchable fields, the system will search through its database to find any matches or close matches to the account number, account status, name, address, phone, email, or date of birth. This operation can take some time, depending on your search query. In general, the more specific you are, the shorter the search time will be. So, if you search "A," the search will take longer than if you search "Apple."

    You can also select the fields that you search by. If you search by account number, you'll need to format your account number as follows: AA-BB-CC. Do not use asterisks, and do not use any characters other than numbers and dashes. If you search by address, format your address as follows: XX Street Nm. Specific searches should be relatively fast, regardless of search query.

    You can also display all of the entries in the database. This may take a very, very long time (upwards of an hour for large databases!), and your web browser may become unresponsive or appear to hang. Due to the nature of this search type, the system does not permit you to style the page; instead, you can only view a plaintext version.
